devflow-cli 1.5.0 Release Notes

We're pleased to share version 1.5.0 of devflow-cli, a focused update that improves reliability and strengthens support for modern development setups. This release is all about making the tool work more smoothly out of the box, with fewer interruptions to your workflow.

- More reliable template loading: devflow-cli is now smarter about finding your project templates, even when they're stored in non-standard locations. This means fewer frustrating "template not found" errors and a smoother experience when setting up or running projects in varied environments.

- Better support for TypeScript and React projects: The tool now includes built-in code quality checks tailored specifically for TypeScript and React codebases. Teams working with these technologies will benefit from more accurate feedback and fewer false warnings when validating their projects.

- Improved release confidence: Our internal build process now automatically runs the full test suite after each build, helping us catch issues earlier and deliver more stable updates to you.

As always, we appreciate your feedback — it helps us continue improving devflow-cli with every release.
